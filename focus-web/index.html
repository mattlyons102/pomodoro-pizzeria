🍅 How the timer works

Work in focused "bakes," then take a short "cool" break — repeat to stay fresh.

  • Start / Pause begins or holds the current timer; Reset clears everything back to Round 1.
  • It auto-flips: a focus bake → a cool break → the next bake, dinging at each switch.
  • Every 4th pizza earns a longer Wood-fired rest 🔥 instead of a short break.
  • Tweak Bake / Cool / Long rest minutes (while paused) to set your own rhythm.
